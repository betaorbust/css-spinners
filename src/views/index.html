CSS Loading Spinners

The top Code Pen from 2014 as 8 animaged SVG loading spinners. Unfortunately, they don't work on IE -- even 10+. So here are the same spinners, rewritten in pure HTML and CSS. They're efficient (using transforms for speed where possible) and work in IE.
